Projeto acadêmico desenvolvido para a disciplina de **UPx** no curso de Engenharia de Computação da **FACENS**.

O sistema consiste em um **alimentador automático de animais**, integrando um protótipo físico com **ESP32** e uma aplicação web para controle e monitoramento da alimentação dos pets.

---

## 🚀 Tecnologias Utilizadas

- **Frontend:** React.js  
- **Backend:** Django  
- **Banco de Dados:** PostgreSQL  
- **Infraestrutura:** Docker & Docker Compose  
- **Hardware:** ESP32  
- **Componentes:** Sensor de peso e Servo motor  

---

## 🛠️ Funcionalidades

- Liberação automática de ração  
- Controle de quantidade por animal  
- Cálculo baseado em idade e nível de atividade  
- Dashboard com histórico de consumo  
- Integração entre sistema web e ESP32
